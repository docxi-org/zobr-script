# Подключение к ZS

ZS — это MCP-сервер. Для работы нужен AI-ассистент, поддерживающий удалённые MCP-подключения по HTTPS. Адрес сервера:

```
https://zs.docxi.org/mcp
```

Сервер использует **OAuth 2.1** для аутентификации. При первом подключении клиент откроет страницу входа, где вы введёте логин и пароль. Далее сессия поддерживается автоматически.

## claude.ai

Доступно на тарифах Pro, Team и Enterprise.

1. Откройте **Settings** (иконка профиля в левом нижнем углу)
2. Перейдите в **Connectors**
3. Нажмите **Add custom connector**
4. Введите адрес сервера: `https://zs.docxi.org/mcp`
5. Нажмите **Add** — откроется окно браузера со страницей входа ZS
6. Введите email и пароль, нажмите **Sign in**
7. Коннектор появится в списке — он синхронизируется между Claude Desktop, iOS и Android

После подключения инструменты ZS (`zs_register`, `zs_start` и др.) доступны в каждом разговоре. Попросите Claude зарегистрироваться и запустить скрипт.

## ChatGPT

Доступно на тарифах Plus, Pro, Team и Enterprise. Требуется Developer Mode.

1. Откройте **Settings** → **Apps** → **Advanced settings**
2. Включите **Developer Mode**
3. Перейдите в **Connectors**, нажмите **Create**
4. Введите название (например, «ZS»), описание и адрес сервера: `https://zs.docxi.org/mcp`
5. Выберите тип аутентификации: **OAuth**
6. Сохраните коннектор
7. В новом чате нажмите **+** → **More** → выберите коннектор ZS для активации

Действия записи требуют подтверждения пользователя в каждом чате.

## Cursor

Доступно в Cursor v0.48+.

1. Откройте **Settings** → **MCP Servers**
2. Нажмите **Add** или отредактируйте `.cursor/mcp.json`:
   ```json
   {
     "mcpServers": {
       "zs": { "url": "https://zs.docxi.org/mcp" }
     }
   }
   ```
3. OAuth-поток откроется в браузере автоматически при первом использовании

## Windsurf

1. Откройте панель **Cascade** → нажмите иконку **MCP**
2. Добавьте сервер или отредактируйте `~/.codeium/windsurf/mcp_config.json`:
   ```json
   {
     "mcpServers": {
       "zs": { "serverUrl": "https://zs.docxi.org/mcp" }
     }
   }
   ```

## Claude Code (CLI)

Если вы работаете в Claude Code в этом репозитории, ZS уже настроен как локальный MCP-сервер в настройках проекта. Дополнительная настройка не нужна — инструменты доступны сразу после `/mcp` реконнекта.

## После подключения

Типичный сценарий работы:

1. Попросите ассистента **зарегистрироваться** — он вызовет `zs_register` и получит системный guide
2. Попросите **показать скрипты** — `zs_list` покажет доступные скрипты
3. Попросите **запустить скрипт** — например, «запусти скрипт insight на нашем модуле аутентификации»

Ассистент работает с MCP-протоколом автоматически. Вы общаетесь на естественном языке.
